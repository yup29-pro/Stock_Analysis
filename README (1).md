# 📈 Stock_Analysis — Market Trend Analyzer

> Real-time stock data analysis with interactive charts & volatility tracking!

---

## 🚀 What is this?

Ever wondered how **AAPL, TSLA, GOOGL, MSFT & AMZN** are performing in the market?
This tool downloads **real stock data**, crunches the numbers, and gives you beautiful interactive charts — all in just a few seconds! 🎯

No finance degree needed. Just run it and explore! 😄

---

## ✨ Features

- 📊 **Live Stock Data** — Pulls real market data from Yahoo Finance (2023–2025)
- 📉 **Price vs Moving Averages** — See 20-day & 50-day trends clearly
- 🔥 **Volatility Tracker** — Compare how risky each stock is vs the others
- 🌙 **Dark Mode Charts** — Because light mode is overrated 😎
- ⚡ **Interactive Charts** — Hover, zoom, pan — fully interactive!

---

## 🛠️ Technologies Used

| Tool | What it does |
|------|-------------|
| 🐍 Python | Core programming language |
| 📦 yfinance | Downloads real stock market data |
| 🐼 Pandas | Cleans and organizes the data |
| 📊 Plotly | Creates beautiful interactive charts |

---

## ⚙️ How to Install & Run

### Step 1 — Download the code
Click the green **Code** button → **Download ZIP** → Extract it

### Step 2 — Install required libraries
Open terminal / VS Code and run:
```bash
pip install yfinance pandas plotly
```

### Step 3 — Run the code
```bash
python Stock_Analysis.py
```

### Step 4 — Pick your stock!
```
Enter stock name (AAPL / TSLA / GOOGL / MSFT / AMZN): TSLA
```
And boom — your charts appear! 🎉

---

## 📸 What You'll See

### Chart 1 — Price vs Moving Averages
- 🔵 **Blue line** — Actual stock price
- 🟠 **Orange dashed** — 20-day Moving Average (short term trend)
- 🔴 **Red dotted** — 50-day Moving Average (long term trend)

### Chart 2 — Volatility Comparison
- See how much your chosen stock swings vs all others
- Higher volatility = higher risk 📊
